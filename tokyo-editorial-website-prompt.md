 # Tokyo Editorial Scroll Experience — Antigravity Prompt

Create an award-winning interactive website inspired by modern editorial magazines, Apple product pages, and Awwwards-winning creative portfolios.

Use the uploaded Tokyo Tower poster as the primary visual reference for layout, typography, composition, color palette, and visual language.

---

## Overall Style

* Vintage Japanese editorial design
* Minimalist luxury aesthetic
* Large bold typography
* Massive headlines occupying 60–80% of the viewport
* Soft paper texture background
* Film grain overlay
* Muted cream, sky blue, rust orange, and deep navy palette
* Magazine-inspired layouts
* Premium motion design
* Clean grid system
* High contrast typography
